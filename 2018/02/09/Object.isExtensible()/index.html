Object.isExtensible()

发表于 2018-02-09 | 分类于 Diary

Object.isExtensible(obj) 判断对象是否可扩展
@argm 要检测的对象
@return Boolean 是否可扩展
默认情况下，对象是可扩展的：即可以为他们添加新的属性。以及它们的 __proto__ 属性可以被更改

DEMO

注意

在 ES5 中，如果参数不是一个对象类型，将抛出一个 TypeError 异常。在 ES6 中， non-object 参数将被视为一个不可扩展的普通对象，因此会返回 false 。

Object.seal()

发表于 2018-02-09 | 分类于 Diary

Object.seal(obj) 密封一个对象，并返回被密封后的对象。
密封对象将会阻止向对象添加新的属性，
并且会将所有已有属性的可配置性（configurable）置为不可配置（false）。
即属性变的不可删除，
数据属性不能被重新定义成为访问器属性，严格模式下会报 TypeError
但是可写性描述（writable）为可写（true）的属性的值仍然可以被修改。

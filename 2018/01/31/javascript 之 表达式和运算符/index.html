javascript 之 表达式和运算符

发表于 2018-01-31

表达式和运算符

  1. 表达式
    • 原始表达式
      • 常量、直接量 3.14, "str"
      • 关键字 var
      • 变量 i,j,k
    • 复合表达式
      • 原始表达式 + 运算符 3*a
    • 数组、对象初始化表达式
      • [1,2] 等价于 newArray(1,2)
      • [1,,2] 等价于 [1,undefined,2]
      • {x:1,y:2} 等价于 var o = new Object();o.x=1;o.y=2;
    • 属性访问表达式
      • o.x
    • 函数表达式
      • var fn = function(){} function fn(){}
    • 调用表达式
      • fn()
    • 对象创建表达式
      • new Fn('1') new Object 创建一个空对象 //{}

  2. 运算符

    • 一元运算符 -a、+a、++a、--a、void
    • 二元运算符 a+b

    • [ ] 三元运算符 true?a:b

    • [ ] 赋值运算符 =

    • 赋值运算符 |、&、!
    • 算术运算符 +、-、*、/ ……
    • 比较运算符 == / === / > / < ……
    • 逗号运算符 var val = (1,2,3) //val = 3
    • delete 运算符（delete 是否可以成功删除属性与 Object.defineProperty 中对象属性设置有关。具体参看：Object.defineProperty)

var let const 区别

js 三种定义变量的方式，分别是 var/let/const

  • const 定义的变量必须初始化，不可以修改。

  • let 声明块级作用域被限制的本地变量

    javascript {} 是没有作用域的限制的，但如果用 let 声明就有了相应的概念。
    let 可以只声明，不赋初值，初值默认是 undefined

  • var 同样可以只声明，不赋初值，初值默认为 undefined，与 let 相反，没有{}作用域的限制。

关于 promise 函数的面试题

打印结果：2 4 1 3
解析：setTimeout 与 then 都是异步执行的，按照代码的执行顺序先实例化 Promise，同时执行回调打印 2 ，异步需要时间，接着打印 4，setTimeout ，延时更长一些。
```javascript
setTimeout(()=>{
console.log(1);
})

new Promise((resolve,reject)=>{
console.log(2);
resolve(3);
}).then(function(n){
console.log(n);
})

console.log(4);
// 2
// 4
// 3
// 1
```
